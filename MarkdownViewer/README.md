# SimpleInspector Markdown Viewer

## Overview

`MarkdownViewer` is the module for browsing markdown documents inside Unity.
It indexes markdown files and displays them in a dedicated split-view editor window.

## Contents

- `Editor`
  markdown index builder, parser, menu entry, and viewer window
- `Examples`
  example markdown documents and viewer samples

## Menu

- `SimpleInspector/Markdown Viewer`

## What It Does

- indexes markdown files from supported project locations
- renders markdown content inside Unity
- supports navigation through a tree view
- allows split-view browsing similar to other SimpleInspector tools
